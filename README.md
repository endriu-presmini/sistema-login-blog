# 🔐 Sistema Login + Cadastro + Blog

Interface web completa com tela de login, cadastro de usuário e página de blog, desenvolvida com HTML e CSS puro.

## 📸 Páginas

- **Login** — autenticação com e-mail e senha
- **Cadastro** — criação de conta com validação de campos
- **Blog** — listagem de artigos com cards estilizados

## 🚀 Tecnologias

- HTML5
- CSS3 (variáveis CSS, flexbox, animações, media queries)

## ✨ Destaques

- Tema escuro com paleta de cores elegante
- Animação de entrada nos cards (`fadeUp`)
- Navbar fixa com scroll
- Layout totalmente responsivo
- CSS Variables para consistência visual

## 📁 Estrutura

```
├── login.html
├── stylelogin.css
├── cadastro.html
├── stylecad.css
├── tela1.html
└── styletela1.css
```
